## Run Locally

Clone the project

```bash
  git clone https://github.com/Eth3ri0n/Diane.git
```

Go to the project directory

```bash
  cd Diane
```

Install dependencies

```bash
  npm install
```

Start the Tailwindcss

```bash
  npm run start
```


## Tech Stack

**Client:** HTML, CSS, TailwindCSS

**Server:** Node
